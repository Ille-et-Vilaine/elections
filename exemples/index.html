<script>(function(d, s, id) { var js, fjs = d.getElementsByTagName(s)[0]; if (d.getElementById(id)) return; js = d.createElement(s); js.id = id; js.async=true; js.src = "http://makinacorpus.github.io/elections/iframe-loader/mkc.js"; fjs.parentNode.insertBefore(js, fjs); }(document, 'script', 'mkc-iframe-loader'));</script>

Options d'intégration

Voici les différentes options disponibles. Pour les utiliser, il suffit d'ajouter les attributs correspondant sur la balise :

Choix de la Hauteur : data-height="pourcentage"

La valeur représente la hauteur en pourcentage relative à la largeur de l'élément. Cela vous permet de définir un ratio de carte. Par défaut, la hauteur représente 75% de la largeur.

<div class="mkc-placeholder" data-source="2015-resultats-departementales" data-dpt="31 data-height="30"></div>

Affichage d'un logo : data-logo="url_du_le_logo"

Vous avez la possibilité de préciser un lien vers une image qui sera ajoutée dynamiquement au bas de la carte (par exemple, ajouter un logo)
